# 視点表現のための二部グラフに基づいた双対空間への埋め込み

視点を考慮した二部グラフ埋め込み手法の実装

## 概要

本リポジトリは、修士論文「視点表現のための二部グラフに基づいた双対空間への埋め込み」で提案した埋め込み手法の実装です。

従来の埋め込み手法では、データの関係性を固定的に扱うため、視点の違いを適切に反映することが困難でした。本手法では、視点を数学的に定義し、「視点はアフィン部分空間である」という理論に基づいた埋め込み手法を提案します。

### 主な特徴

- **視点の数学的定義**: 視点をアフィン部分空間として定式化
- **双対空間への埋め込み**: オブジェクト空間とタグ空間の双対関係を保証
- **理論的に導出された目的関数**: 確率モデル（ガウス分布）から最尤推定により導出
- **2つの射影機能**:
  - **余射影（視点の選択）**: 特定の視点から見たデータ表現を取得
  - **正射影（視点の変換）**: 特定の視点を除去したデータ表現を取得

## 背景・動機

現代のデータ環境において、同じデータであっても異なる視点から見ることで解釈が変わることが多く、視点に応じたデータの適応的な表現が求められています。

例えば、音楽推薦システムでは：
- 「ジャンル」という視点で曲を探すユーザー
- 「気分」という視点で曲を探すユーザー

同じ楽曲でも、視点によって類似度が変化すべきです。本手法は、このような視点の多様性を明示的にモデル化します。

## 理論的背景

### 視点の定義

視点 v ∈ V を選択すると、オブジェクト集合 U に対して同値関係が定義され、商集合 U/v を構成します。

### 双対アフィン部分空間

オブジェクトの埋め込みベクトル u と タグの埋め込みベクトル v は、双対関係を満たします：
```
u^⊤v = 1
```

この制約により、オブジェクト空間とタグ空間が双対な関係となり、視点ごとの整合的な表現が可能になります。

### 目的関数

確率モデル（ガウス分布）を仮定し、最尤推定により以下の目的関数を導出：
```
F[U,V] = Σ[x_ij {β/2(u^⊤v - 1)² - α/2·cos²(u,v)} 
         - (1-x_ij)log(1 - e^(-β/2(u^⊤v-1)²))]
```

詳細は論文を参照してください。

## 実験結果

### 動物データ（Kohonen データセット）

- **データ**: 17種の動物 × 21個の特徴タグ
- **結果**: 
  - 「肉食/草食」「哺乳類/鳥類」という複数の視点を同時に学習
  - 視点の切り替えにより、分類が動的に変化することを確認
  - 従来手法と同等以上の埋め込み性能

### 人工データ（3クラスメタタグ）

- **データ**: 30種の動物 × 25個のタグ（哺乳類/鳥類/爬虫類 × 肉食/草食）
- **結果**:
  - 従来手法では困難だった3クラスの動物種を適切に分離
  - メタタグ間の直交関係を正しく学習

## 従来手法との比較

### 既存の二部グラフ埋め込み手法との違い

| 手法 | 視点の扱い | 目的関数 | 多クラス対応 |
|------|-----------|---------|-------------|
| Two-Tower | 固定 | 経験的設計 | 制限あり |
| 従来手法(前任者) | 暗黙的 | 経験的設計 | 2クラスまで |
| **本手法** | **明示的** | **理論的導出** | **3クラス以上** |

### 主な優位性

1. **視点の明示的モデル化**: 視点を数学的に定義し、動的に切り替え可能
2. **理論的整合性**: 確率モデルから目的関数を導出、双対性を保証
3. **多クラス対応**: 3クラス以上のメタタグ構造にも対応
4. **2つの射影機能**: 余射影と正射影による柔軟なデータ分析
